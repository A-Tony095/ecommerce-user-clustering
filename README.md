# 电商用户行为聚类分析（K-Means 无监督学习）

## 项目介绍
基于电商用户行为数据，使用 Python 实现 K-Means 聚类完成用户分层，实现精准营销分析。

## 项目流程
1. 大数据分块读取
2. 数据去重、时间筛选、数据清洗
3. 构建用户行为特征（浏览、加购、收藏、购买）
4. 特征标准化（StandardScaler）
5. 肘部法则确定最佳 K 值
6. K-Means 聚类模型训练
7. 轮廓系数评估聚类效果
8. 可视化展示用户分层
9. 用户画像分析与营销建议

## 技术栈
- Python
- Pandas
- Matplotlib
- Scikit-learn
- K-Means 聚类

## 项目结果
- 将用户划分为 4 类价值群体
- 轮廓系数：0.346
- 完成用户分层与精准营销方案
